Steps to running the client:

1. Download the client python file.
2. Open up a terminal and navigate to the directory where the client file is located.
3. Run the client file by typing `Python clientSide.py localhost 5000` in the terminal.
    a. The first argument is the IP address of the server
    b. The second argument is the port number that the server is listening on.
    c. Both arguments are required.
4. The client is now connected to the server and can send and receive messages.

Running multiple clients:
1. Follow the steps above to run the client file.
2. Each new client will need a new terminal window.
3. All client must connect to the same IP address and port number.
