Steps to running the server:

1. Download the server python file.
2. Open up a terminal and navigate to the directory where the server file is located.
3. Run the server file by typing `Python serverSide.py 0.0.0.0 5000` in the terminal.
    a. The first argument is the IP address of the server
    b. The second argument is the port number that the server will be listening on.
    c. Both arguments are required.

The server is now running and listening for incoming connections.
